Gits and How They Work


Git is a software that tracks and manages source code history. It is very useful when coding because if you mess up and say accidently deleted all your code and saved it, git can revert the file back to a previous version. It's also useful when collaborating, as each person can branch off the original code and put changes without changing the main branch.

Branches

Gits work by having one main master timeline. This is where you can branch off, commit changes, and merge branches; it's your main file. You can branch off of the main branch and make changes to the code without affecting your original source, then, if you're happy, you can merge the branch back with the master timeline. It's important that 2 or more people are not working on the same file on different branches, otherwise, you could get a merging error.

How to Install Git

You can download git from here. If you're using the CS50 ide, it is already installed. Once you have it installed, open your terminal and input the commands:$ git config --global user.name "FirstName LastName" and $ git config --global user.email "email@example.com".

Git Commands

  • git init - This will initialize an empty repository in your current directory.
  • git add (file) - This will add any file you input to the staging area, alternativly you can use .. to add all files in your current directory to the staging area.
  • git status - This command will show all the files in the staging area.
  • git commit -m"message" - This will commit your changes (files in the staging area) on your branch (think of it as saving your work) followed by a message on what was changed.
  • git log - This will show a log of all changes made to the master branch.
  • git checkout -b (branch name) - This command will create a new branch off of the master branch. You can use add and commit commands on this branch just as done on the master branch.
  • git checkout (branch name) - This will allow you to move through your different branches.
  • git branch - This lists a log of all branches you currently have.
