MiCard


On the right is the final product of the project MiCard. MiCard was essentialy a business card that was made in Flutter. It uses a series of rows, columns, and containers to build what you see. Although you can't see it in this image, there is also a button on the lower right that takes the user to an about me page. This project was uploaded on GitHub Pages

Pseudocode

Create an app that displays a circle avatar image, your name, your position, your phone number, and your email all stacked on top of each other. Create a second page with a working button.

Challenges

The first challenge I had with this project was stretching the white rectangles across the screen without an overflow error. I resolved this by putting colored containers in rows instead of by themselves.

The main issue that I had making this project was getting the button to a second page to work. The issue at first was it couldn't grab the context as the code was in the wrong spot. Then, the second issue was the button simply wouldn't work and would output a navigation error. The problem was resolved by creating a new class called MyHomePage in a stateful widget.
