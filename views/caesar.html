Caesar


A Caesar cipher is a cipher that shifts letters by a certain amount of times down the alphabet. The goal of this project is to make a program that can encrypt text with any given key, but keep capitals and punctuation. For this, we will need a key and text from the user, which will then be used in a function that will shift the ASCII values depending on the key inputted by the user. So, for this to work, we will need the user to input in the terminal their key, and then store it as a string. Then, we will need the program to reject input that is not an integer or if there are too many arguments (to prevent a segmentation fault). This can be done by using an if else statement and using an exit code to kill the program and output an error message if the user failed to comply. Now, we need to prompt the user for text and store it as a string.

Then we're going to have to create a function that will be called over and over again that will convert the char value in a string to an int ASCII value and shift it by the key amount, then convert it back to a char. Now that we have a function to do this, we need to use a for loop to call it for each and every char in our string length array. We can use the function atoi() to convert the key argument to an integer. After the key and char are passed into the function, take the return value and store it into an integer that will then be printed to the terminal. Now, it is important that we define our constant number at the top for the ASCII base values, 65 being for uppercase and 97 being for lowercase. It is also important to note that our encrpyt function ignores punctuation, this can be done by having if/else statements in our encrypt function that checks to see if it is a letter (Using the functions isupper and islower to check for uppercase and lowercase letters).

The Libraries

Libraries are pre-written code that we can use in our programs that can be useful. This project uses many functions that require certain libraries to use (such as the atoi() function). These libraries and the associated functions used in this project are as follows:

  • stdio.h - This is the Standard Input and Output library. Most C programs use this library.
  • cs50.h - This is the CS50 library. It contains useful datatypes such as the string datatype.
  • ctype.h - This library contains the function isalpha(), islower(), and isupper().
  • stdlib.h - This library is the C Standard Library library. It contains the function atoi().
  • string.h - This library is the string library that contains the function strlen().
  • Although not used in this project, a header file could be made that contains the encrypt() function. This would negate the need to define the function in the project, but you will need to include it at the top.

The Code:

Pseudo Code:

  • Make a function that will take a character and a key that will shift the characters ASCII value by the amount of the key without affecting capitals or non alphabetic characters. Make sure to wrap around by using modulo (%) 26.
  • Get a key value from the user by using an argument in the command terminal and store it as a string. If they do not cooperate, display and error with instructions for how to use the progam properly. Stop the program.
  • Prompt the user for a string value and store it in a string variable called plaintext.
  • Make a loop that will iterate through every character in the string and pass it through the encrypt function along with the key value which should be converted to an integer.
  • Print each character that the function returns.
