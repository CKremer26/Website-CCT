Mario


In the game Mario, toward the end of level 1-1, Mario ascends a right-aligned pyramid to reach the flagpole. The goal of the program "Mario", is to have a user input an integer (and ONLY an integer) between 1-8 and have the terminal print out a right-aligned pyramid with the corresponding amount of hashes.


First, we need to make it so that the program prompts the user for an integer. This can be done by using the get_int function and then store that value in a variable called "height". However, the program should only take an integer between 1 and 8. If the user fails to comply, they should be re-prompted, this can be done with a Do While Loop. Then, using the integer that we got from the user, the program needs to print out the corresponding amount of hashes for the first line, this can be done using a For Loop. However, now we needs to print the correct amount of lines of hashes, this is where a Nested for loop comes in (basically putting a for loop within a for loop). However, we also need this pyramid to be right-aligned. This means that we also need the program to print the correct amount of spaces before printing the hashtags.
