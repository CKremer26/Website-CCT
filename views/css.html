Cascading Style Sheets


Cascading Style Sheets, or more commonly known as CSS, is a language that is used in conjunction with HTML for styling purposes. CSS is useful when changing the font, fontsize, color, or margins in text. As well, it is used to change the background color of your page, and if you want to bold or underline your text (alternativly, you can use the <b> and <u> tags). It is also very useful when formating your website when using it with <div> tags. There are many different ways that you can use CSS in html, these include: inline styling, IDing an element, creating a class, or using an external CSS file.

You can also use different color values in CSS. These can be implemented for a background or color of text. Accepted formats for color are: RGB values (R, G, B, up to 255), Hexadecimal (#FFFFFF), or a list of color names (case sensitive).

Inline Styling

CSS inline styling involves using the style attribute in the beginning tag of an element. Typically used in the format of style="css code goes here". The main problem with this form of styling is your code tends to get really messy and unorganized. To stylize certain parts of text, the <span> tag can be used.

IDing an Element

To ID an element, you use the id="idname" attribute in the first tag of an element. Then, at the head section of your page, create a <style> tag, and inside it, define the id name using #idname. Then, in curly braces {} you can put in your css code. Example: #ParagraphOne {color: white; font-size: 25px;}. Note: this can only be done for ONE element.

Classes

Classes are useful when you want to change a group of elements in a page. To do this, you do the same thing as IDing, but, instead of a #idname, you do a element.classname{}. Example: p.paragraph {font-size: 20px;}. Then, to use this, use the class="classnames" attribute in the first tag of the element defined in the class. You can assign multiple classes to an element.

External CSS

External CSS is having a seperate file which contains all of your sites styling code. The CSS file can be written in any text editor, but must not have any HTML code and must have the file extension of .css. Because this is a seperate file, you will have to import the style sheet. This can be done by using the <link> tag in the header section of your site. The attributes to go with it is rel (relationship), and href (hypertext reference). Would look something like this: <link rel="stylesheet" href="(name of css file).css">. External CSS is useful because it keeps all of your CSS code organized, meaning if you want to change something you don't have to search through all of your HTML code.
