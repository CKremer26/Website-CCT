What is GitHub pages?


GitHub pages is a service provided by GitHub that builds and deploys your repository. In this class, we used it to deploy our MiCard Flutter projects. You can find the deployment of my app here.

How to Use GitHub Pages

  • Go to GitHub and create a new repo, copy the link for later.
  • In your Flutter app in VS Code, navigate to the web folder, then open index.html. Remove the base HREF line.
  • In the terminal, run the command $ flutter build web. Once it's finished, change your directory to the newly create build folder.
  • Initialize a git repo ($ git init). Connect that repo to your GitHub repo.
  • Add everything to the staging area ($ git add .) and commit the changes. Push the files to your GitHub repo.
  • Now, back in GitHub, open the index.html folder and delete the base HREF line on line 17. Commit the change.
  • Go to repo settings --> pages. Change the branch to main and save the changes. Wait for your site to deploy (may take a while).
  • Your website should now be live.
