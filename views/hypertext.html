Hypertext Markup Language


Hypertext Markup Language, or more commonly known as HTML, is a coding language that most of the web is built off of. In fact, this entire website is built off of HTML! There are many different things that you can do in html that can build a website, such as making text, showing images, or even videos! You can also inbed your own scripts in your site if you wanted to. You can write HTML code in any text editor with the extension of .html and they have to start with <!DOCTYPE html> (declares the document type). However, to make this website, I used a cloud based editor called Glitch.

The Layout

A html webpage is usually set up into three sections: a <head>, <body>, and optional <footer>. In the head section, you can add <title> a which will put text on your tab at the top. The body section is typically where the main part of your webpage will go. Here, if you want to add a title, you can use <h1> tag. This will put a header at the top of your screen. If you want to write a paragraph, you can use the <p> tag. another important tag to mention is the <div> tag. This is a divider tag, and it is useful when applying a certain layout or styling to your website.

Other useful tags and elements

  • <a> tag. This tag is an attribute tag. This tag is mainly used with the href (hypertext reference) attribute which creates a hyperlink. A hyperlink redirects you to another page or website when clicked.
  • <img> tag. This is an image tag. It allows you to put images on your page. Must include a src (source) attribute. You may also use the width and length attribute to customize the size of the image. If you want, you can add an alt (alternate) attribute which displays text if the image fails to load.
  • <video> tag. Displays a video. Attributes that can be added is width, height, and controls (gives video controls to the video). Requires a <source> tag which will have a src attribute that links to the video asset.
  • <ul> tag. This tag is for an unordered list, which will be a bulleted list. This is used in conjunction of the <li> tag, which is list items.
  • There are many, many other tags and attributes not mentioned on this list. You can find what they are and how to use them on W3Schools.
