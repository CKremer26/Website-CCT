What is C?


C is a rather old coding language used to code programs to tell them what to do. What is a coding language you may ask? A coding language is a language that is used by people that tells a computer what to do. C is one of the most basic coding languages there are.


IDE

IDE stands for Integrated Development Environment. Basically, it is a software that holds directories, files, and our code. It also consists of a terminal, which is how we navigate our directories and run programs. I am using the CS50 IDE.


Source Code vs. Machine Code

Source code is the written code that you see on the right. Depending on the language you use there will be different syntax. The purpose of source code is so that humans can understand it. Machine code, on the other hand, is code that the computer can understand, and is composed of binary (1s and 0s). Therefore, it is essential to have a compiler that will translate the source code to machine code. In the terminal this can be done with the "make" command.
