Bootstrap


Bootstrap is an open-source stlying resource library, meaning that it's a library that lots of developers have been working on for years. It's used in most modern day websites because it is responsive, meaning it looks appealing and adjusts to the size of your screen. It also allows us to create interactive buttons, dropdowns, and a navbar. You may have noticed already that this website has a navbar, this is thanks to Bootstrap! Bootstrap uses containers and fluid-containers to organize the web page into a grid. This is how Bootstrap is usefull in web page design.

How to Use Bootstrap

There are a couple of ways to import the Bootstrap library. The first is through downloading the library into a folder and linking it locally, like with external CSS. Or, more commonly done, you can link it through a Content Delivery Network (CDN), which is a URL that links your page to the library in the cloud. This is done by putting a link in your header section. You can find a good tutorial on how to import the Bootstrap library here.

Uses of Bootstrap

  • Jumbotron - You can create a jumbotron using Bootstrap, which is basically a box that helps text pop out. You do this by giving a div element the class of "jumbotron".
    An example of a jumbotron
  • Navbar - You can create navbars in your page through Bootstrap by using the <nav> element. Navbars are useful for navigating a web page and the bar at the top of this page is an example of a more complex navbar.
  • Dropdown Menus - You can create dropdown menus using a series of classes and data toggles in a button with Bootstrap. This allows for nice and organized lists that won't take up the entire screen.

You can find more ways to use bootstrap here.
